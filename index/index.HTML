Juego ping pong en Java Script

*Presiona espacio para iniciar o pausar el juego*

Por: Julian Avila. TQA SofkaU

Instrucciones de juego:

* El jugador 1 mueve la barra izquierda con la tecla (W) para subir y tecla (S) para bajar.

* El jugador 2 mueve la barra derecha con fecha arriba y la fecha abajo.

* Si la bola toca las paredes verticales se acaba el juego y debe dar click en "Jugar de nuevo".
